DEFINIÇÕES DA APLICAÇÃO

TELA DE LOGIN
- Será lidada pelo sistema de autenticação do Firebase

CADASTRO DE USUÁRIOS
	Dados Requisitados:
		O cartão utilizado para o cadastro já existe no banco? (bool)
		- Comparar código do cartão com todos os 'cd_cadastro' no banco
	Dados Enviados:
		Nome de cadastro
		Imagem de perfil
		Código do cartão que vai ser cadastrado
		- O código vai ser enviado em 2 rotas diferentes: uma para consulta e outra para o cadastro

SETORES
	Dados Requisitados:
		Todos os setores no banco
		ID de cada setor
		Nome de cada setor
		Número total de pessoas por setor
		
CADASTRO DE SETORES
	Dados Requisitados:
		
MAPA DO SETOR
- Acessado ao clicar na etiqueta do setor que se deseja visualizar
	Dados Requisitados:
		Nome da localização
		Número do andar
		Todas as pessoas naquele setor
		Imagem de perfil de cada pessoa
		ID por pessoa para evitar usar imagem de perfil para a busca das informações
	Dados Enviados:
		Setor ao qual se deseja visiualizar informações
		
INFORMAÇÕES POR PESSOA
- Acessadas ao clicar no ícone da pessoa dentro do setor
	Dados Requisitados:
		Nome de cadastro
		Horário de entrada
	Dados Enviados:
		ID da pessoa à qual se deseja visualizar o cadastro
